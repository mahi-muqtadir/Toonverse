Introduction:

Our project “Toonverse” is inspired from our childhood cartoons such as Mickey Mouse, SpongeBob SquarePants, Pokemon, Up, Dexter Laboratory, The Bare Bears, The Powerpuff Girls , Phineas and Ferb, Doremon and The Simpsons.
We tried to make a multiverse of cartoons using characters from different cartoon shows and also used a Hot Wheels car design called “Pixel Shaker”. There are 3 scenes on our project  and the whole project was implemented
using C++,GLUT and OpenGL.  

Objective :

The main objectives are, 
i. To understand the basic concepts and working process of OpenGL and GLUT. 
ii. To implement algorithms such as Digital Differential Analyzer (DDA),Bresenham’s Mid           
Line and Mid Point Circle. 
iii. To implement 2D scenes and a 3D object. 
iv.To implement Transformation (Translate, Scaling, Rotation) and Timer function. 

Output  

The output shows a 2D morning scene where clouds and balloon moving in the sky . A car running on the road and characters jump when clicked on “SPACEBAR”. Also there is a 3D brick near the road. The scenery changes if  ‘2’
or ‘3’ is pressed. On the 2nd scenery there is SpongeBOB floating under the water which can be move or controlled by w,s,a,d. There is a moving submarine and bubbles.
